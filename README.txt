README

1. 프로그램 설명
음원을 관리할 수 있는 관리자모드와 음악을 플레이, 검색할 수 있는 유저모드로 구분지은 음원관리프로그램 및 뮤직 플레이어

2. 프로그램 작동 방법

(i) 유저모드

1) 회원가입버튼을 눌러 회원가입한다.
2) 로그인버튼을 눌러 로그인을 한 뒤 유저모드로 들어간다.
3-1) 곡 검색

검색할 카테고리를 콤보박스에서 선택한 뒤, 검색할 내용을 입력 후 검색 버튼을 누른다.
다시 원래 곡 목록을 보기 위해서는 카테고리를 Search로 선택한 뒤 검색 버튼을 누른다.

3-2) 플레이리스트에 곡 추가

곡 목록에서 추가버튼을 누른 뒤 확인을 누른다.

3-3) 플레이리스트 곡 재생

플레이리스트 아래 재생 버튼을 누른다.
<같은 폴더에 포함된 mp3파일만 재생할 수 있으므로 현재는 곡 목록에 저장된 너를만나를 제외한 모든 팝송이 재생됨>
* 한곡만 재생가능 

3-4) 플레이리스트 곡 중단

플레이리스트 아래 중단 버튼을 누른 뒤 Ok를 누른다.


(ii) 관리자모드

1) 관리자번호 1234를 입력해 관리자인증을 완료한다.
2-1) 곡 파일 불러오기

곡 파일 불러오기를 누르면 기존에 있던 곡 파일을 읽어들인다.

2-2) 곡 추가

곡 추가 버튼을 누른 뒤 *고유번호는 입력하지 않고* 나머지 정보를 모두 채운 뒤 확인을 누른다.

2-3) 곡 수정

곡 수정 버튼을 누른 뒤 수정할 곡의 *고유번호를 입력하고* 나머지 정보를 수정하여 확인을 누른다.

2-4) 선택 곡 삭제

삭제하고 싶은 곡의 체크박스를 체크한 뒤, 선택 곡 삭제 버튼을 누른다.

2-5) 전체 곡 삭제

전체 곡 삭제 버튼을 누르면 등록된 모든 곡이 삭제된다.
